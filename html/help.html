Genius Population Options:
New tab when Genius button clicked
New window when Genius button clicked
Populate comment section with Genius lyrics
About:

This extension was developed to assist the music lover to view Genius.com lyric and annotation content with ease while browsing Youtube. To use the extension 3 general options can be used:

  1. Generate New Tab - Will generate a new tab with the Genius.com url to the lyric page for the video.
  2. Split Screen New Window - Will open a new Chrome window with the Genius.com lyrics at half the screen's width as well as shrink Youtube browser window to half the screen's width creating a split-screen effect.
  3. Populate Comment Section - Will re-purpose the comments section of the Youtube video to contain the Genius.com lyrcis with clickable annotations right on the youtube video page!
